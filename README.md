# supermarket-bill-generator
A simple and efficient Python-based billing system designed to simulate a supermarket checkout process.
This project automates item selection, quantity input, price calculation, and generates a final bill for the customer.
**Features**
Add items to the cart
Search items by name or starting letters
Enter quantity and auto-calculate total
Displays itemized bill
Calculates subtotal, tax (if needed), and final amount
Beginner-friendly Python logic
**Technologies Used**:
Python 3
Basic Control Structures
Lists & Dictionaries
User Input Handling
How to Run

**Download the project**:

supermarket.py

Open your terminal or VS Code.

Run:
python supermarket.py

**Learning Outcomes**:

Strengthened Python fundamentals

Improved logic building

Hands-on experience with real-world problem solving

Understanding of interactive text-based applications
